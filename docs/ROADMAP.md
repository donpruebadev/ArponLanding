# ARpón - Hoja de Ruta de Desarrollo Web

## Visión General

**Arpón** es una StartUp de análisis de contratación gubernamental que utiliza inteligencia artificial para detectar irregularidades, opacidad y corrupción en contratos públicos. Nuestro storytelling visual es el de **"cazadores de monstruos"**: cada anomalía es un monstruo que emerge de las profundidades de la burocracia, y Arpón es el arpón que los cazca y expone.

**Tagline**: *"Detectamos lo que otros no ven."*

## Sistema de Monstruos (Storytelling Visual)

### Categorías de Monstruos

Cada monstruo representa un tipo de irregularidad:

| Monstruo | Tipo de Irregularidad | Descripción |
|----------|----------------------|-------------|
| **El Inflado** | Sobreprecios | Contratos con precios inflados artificialmente |
| **La Hidra Burocrática** | Burocracia excesiva | Procesos innecesariamente complejos |
| **El Coloso Blindado** | Falta de transparencia | Entidades que ocultan información |
| **La Sanguijuela** | Desvíos de recursos | Recursos que no llegan a su destino |
| **El Tembloroso** | Miedo a la auditoría | Entidades que evitan el escrutinio |
| **La Serpiente Acorralada** | Conductas evasivas | Intentos de evitar la detección |
| **El Escondido** | Opacidad | Información oculta o difficulte de acceder |
| **El Paranoico** | Resistencia al cambio | Oposición a la transparencia |
| **El Derretido** | Degradación de calidad | Servicios o productos inferiores |
| **El Vacío** | Cumplimiento incompleto | Contratos no ejecutados total o parcialmente |
| **La Ajena** | Intereses externos | Influencia de terceros no declarados |
| **El Declinante** | Deterioro progresivo | Empeoramiento gradual del cumplimiento |
| **El Dormilón** | Inacción | Entidades que no monitorean contratos |
| **La Roca** | Rigidez | Resistencia a adaptarse a nuevas normas |
| **El Letárgico** | Lentitud | Procesos excesivamente lentos |
| **El Archivo** | Burocracia documental | Exceso de papeleo sin sustancia |
| **La Asamblea** | Colusión | Reuniones para coordinar irregularidades |
| **El Plan** | Maquinación | Estrategias elaboradas para evadir controles |
| **El Pacto** | Acuerdos secretos | Convenios entre partes para beneficio mutuo |
| **La Red** | Corrupción sistémica | Redes de corrupción interconectadas |

### Estados de los Monstruos

Los monstruos pueden aparecer en diferentes estados:
- **Victoriosos** - Irregularidades no detectadas
- **Asustados** - Detectados pero no procesados
- **Tristes** - Procesados y sancionados
- **Enfrentados** - Siendo investigados por Arpón
- **Durmiendo/Inactivos** - Potenciales irregularidades
- **Conspirando** - Irregularidades coordinadas

### Elementos Visuales

- **Arpón** (↗): Símbolo de Arpón cazando
- **Ojos cerrados (X)**: Inconsciencia o complicidad
- **Fragmentos de corrupción**: Piezas de evidencia

## Stack Tecnológico Recomendado

### Frontend

- **Framework**: Next.js 14+ (App Router)
- **Styling**: Tailwind CSS + CSS Modules
- **Componentes**: Radix UI + Custom
- **Animaciones**: Framer Motion
- **Gráficos**: Recharts / D3.js
- **Iconos**: Lucide React

### Backend (Futuro)

- **Runtime**: Node.js
- **Framework**: Fastify / NestJS
- **Base de datos**: PostgreSQL + Redis
- **IA**: Python (FastAPI) para modelos
- **Cache**: Redis
- **Queue**: Bull / BullMQ

### DevOps

- **Hosting**: Vercel (frontend) + Railway/Fly.io (backend)
- **CI/CD**: GitHub Actions
- **Monitoreo**: Sentry
- **Analytics**: PostHog
