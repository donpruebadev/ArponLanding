# ARPÓN Landing

Landing page estática generada a partir del diseño de Google Stitch.

## Estructura

- `index.html` — página principal
- `public/logo/` — logo de ARPÓN
- `public/monsters/svg/` — ilustraciones de monstruos
- `stitch_arp_n_cazadores_de_corrupci_n/` — referencia del diseño original de Stitch

## Desarrollo local

```bash
npm run dev
```

Abre http://localhost:3000

## Deploy

Es un sitio estático. Se puede deployar directamente en GitHub Pages, Vercel, Netlify o cualquier CDN.
